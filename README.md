# 01 Client Library

Javascript library for interacting with the 01 Solana program.

## Installation

Using npm
```bash
npm install
```

## Usage example

```python
import foobar

# returns 'words'
foobar.pluralize('word')

# returns 'geese'
foobar.pluralize('goose')

# returns 'phenomenon'
foobar.singularize('phenomena')
```
